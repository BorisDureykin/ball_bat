Игра Ball_Вat

Требования:

При запуске игры отображается игровое поле;
на игровом поле отображается:
    бита;
    мяч;
мяч начинает движение сразу после запуска игры;
бита находится в нижней части игрового поля;
бита перемещается  вслед за движением мыши;

при косании мяча верхней границы игрового поля направление движения мяча по оси Y изменяется на противоположное;
при косании мяча левой или правой границы игрового поля направление движения мяча
    по оси Х изменяется на противоположное;

при косании мяча биты  направление движения мяча изменяется:
    при косании левой части биты направление движения мяча по оси Х "влево";
    при косании правой части биты направление движения мяча по оси Х "вправо";
    при косании центральной части биты направление движения мяча по оси Х не изменяется;

    направления движения мяча по оси Y при косании биты меняется на противоположное;
    изменение угла отражения зависит от места косания биты:
        чем ближе к краю биты:  изменение по оси Y уменьшаются , изменение по оси Х увеличиваются;
        чем ближе к центру биты изменение по оси Y увеличиваются , изменение по оси Х уменьшаются;

в верхней левой части игрового поля отображается количество отбитых мячей;
скорость движения мяча возрастает, а цвет мяча изменяется через каждые 10 отбитых мячей;

Завершение игры:
    если мяч коснулся нижней части поля;
    если отбито 50 мячей;

при завершении игры выводится сообшение что игра завершена и количество отбитых мячей.
